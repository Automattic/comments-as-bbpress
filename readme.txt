A lightweight WordPress plugin that replaces the default comments section of your posts with a bbPress forum.

= Description =

With Comments as bbPress, you can easily create a dedicated forum for each of your WordPress posts. This makes it easier to organize and manage the discussions around your content, while also giving your users a more engaging and interactive experience. You can moderate the comments on your bbPress forums just like you would on regular WordPress comments, and you can use all the bbPress features to enhance your users' experience.

Here are some of the main features of Comments as bbPress:

Replaces the default WordPress comments section with a bbPress forum (requires bbPress 2.6+).
Allows you to use all the bbPress features to enhance your users' experience.
Lightweight and easy to use.

= Installation =

Install and activate bbPress.
Install and activate comments-as-bbpress.
Install and activate the Gutenberg plugin + Blocks Everyhwere (Optional).
Activate the plugin through the 'Plugins' screen in WordPress.
Enjoy!
